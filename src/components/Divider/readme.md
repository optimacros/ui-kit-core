# Divider

<!-- example -->
```jsx 
import { Divider } from 'ui-kit-lite/components/Divider';

const TestComponent = () => (
    <div>
        <Divider />
        <Divider vertical />
    </div>
);
```

## Import
```jsx
import { Divider } from 'ui-kit-lite';
// or
import { Divider } from 'ui-kit-lite/components/Divider';
```

## Properties

| Name       | Type      | Default | Description                          |
|:-----------|:----------|:--------|:-------------------------------------|
| `vertical` | `Boolean` | `false` | If `true`, divider will be vertical. |
